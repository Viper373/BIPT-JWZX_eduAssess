# BIPT-JWZX_eduAssess
### 依赖下载和安装：
利用urllib、subprocess、pathlib等库下载Windows端OCR依赖。

### 登录功能：
使用Selenium库自动填写学号、密码，并利用ddddocr进行识别验证码登录教务在线系统。成功后获取学生信息。对登录过程中可能出现的异常进行了处理，包括验证码识别错误、登录失败等情况。

### 教学评价功能：
登录成功后，使用Selenium模拟用户对未评估的课程进行评价。对每个未评估的课程填写评价建议并提交，处理弹窗确认。如果课程已经评价过，则不再重复评价。

### 进度展示：
使用了tqdm库来展示教学评价的进度，以便用户能够清晰地了解评价过程的进行情况。
